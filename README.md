# Benefit Plan Standard Documentation

This repository hosts the public documentation site for the **Benefit Plan Standard**.  
It is intended for developers, carriers, benefit administrators, and others who wish to understand and adopt the normalized schema for health benefit plans.

The documentation is built using [Docusaurus](https://docusaurus.io) and is published via GitHub Pages.  
It includes the formal specification, field definitions, crosswalk tables, and guidance on adopting the standard.

## Contents

- **docs/** – Markdown source files for the site content.  
- **static/** – Static assets (logo, favicon, images).  
- **docusaurus.config.js** – Site configuration.  
- **sidebars.js** – Sidebar configuration (if used).  
- **package.json** – Node project metadata and scripts (optional).  
- **README.md** – This file.

For more information about the standard itself, see the [Benefit Plan Standard Schema repository](https://github.com/Benefit-Plan-Standard/benefit-plan-schema).
