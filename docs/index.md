# Benefit Plan Standard

Welcome to the **Benefit Plan Standard** documentation site.  
This site provides an overview of the standard, detailed specifications, implementation guidance, and community resources.

## What is the Benefit Plan Standard?

The Benefit Plan Standard is an open, vendor‑neutral schema for normalizing health benefit plan data across carriers.  
It allows plans from different issuers to be represented in a consistent JSON format, enabling interoperability, analytics, and innovation in the benefits ecosystem.
